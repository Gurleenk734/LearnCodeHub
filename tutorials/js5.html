JavaScript Conditions

What are Conditions?

Conditions allow your program to make decisions. Based on true or false, different code will run.

if Statement

if(condition) {
  // code runs if true
}

if-else

if(condition) {
  // true block
} else {
  // false block
}

Example

let age = 18;
if(age >= 18) {
  "Adult"
} else {
  "Minor"
}

else if

Used for multiple conditions:

if(score > 80) {
  "A Grade"
} else if(score > 50) {
  "B Grade"
} else {
  "Fail"
}
