JavaScript Events

What are Events?

Events are actions performed by the user like clicking a button, typing in input, or moving the mouse.

Common Events

  • onclick → when button is clicked
  • oninput → when typing
  • onmouseover → when mouse is over

Example

<button onclick="show()">Click</button> function show() {
  alert("Clicked!");
}

Key Idea

Events connect user actions with JavaScript functions.
