CSS Selectors

What are Selectors?

Selectors are used to target HTML elements so we can style them.

Types of Selectors

  • Element Selector → targets tags
  • Class Selector → uses .classname
  • ID Selector → uses #id

Examples

h1 { color: blue; }

.box { background: yellow; }

#title { font-size: 30px; }

Explanation

  • h1 → styles all h1 elements
  • .box → styles elements with class="box"
  • #title → styles element with id="title"
