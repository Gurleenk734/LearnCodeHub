HTML Tables

What is a Table?

Tables are used to display data in rows and columns.

Basic Structure

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>20</td>
  </tr>
</table>

Tags Used

  • <table> → Table container
  • <tr> → Table row
  • <th> → Header cell
  • <td> → Data cell

Example

<table border="1">
  <tr>
    <th>Name</th>
    <th>Marks</th>
  </tr>
  <tr>
    <td>Gurleen</td>
    <td>90</td>
  </tr>
</table>
