CSS Typography

What is Typography?

Typography controls how text looks on a webpage.

Important Properties

  • font-size → controls size
  • font-family → changes font style
  • font-weight → bold/normal
  • line-height → spacing between lines
p {
 font-size: 20px;
 font-family: Arial;
 font-weight: bold;
}
