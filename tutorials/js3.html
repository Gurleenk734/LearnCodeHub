JavaScript Data Types

What are Data Types?

Data types define what kind of value a variable holds. JavaScript supports different types of data.

Common Data Types

  • String → Text ("Hello")
  • Number → Numbers (10, 3.14)
  • Boolean → true / false
  • Undefined → variable without value
  • Null → empty value

Examples

let name = "Gurleen";
let age = 20;
let isStudent = true;

Check Data Type

You can check type using typeof:

typeof "Hello" → string
typeof 10 → number
