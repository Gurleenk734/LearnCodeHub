HTML Lists

What are Lists?

Lists are used to display items in an organized way.

Types of Lists

  • Unordered List (ul) → bullet points
  • Ordered List (ol) → numbered

Unordered List

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Ordered List

<ol>
  <li>Step 1</li>
  <li>Step 2</li>
</ol>

Example

<ul>
  <li>Apple</li>
  <li>Banana</li>
</ul>
