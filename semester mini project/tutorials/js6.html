JavaScript Loops

What are Loops?

Loops allow you to repeat a block of code multiple times automatically. Instead of writing the same code again and again, we use loops.

for Loop

for(initialization; condition; update) {
  // code runs repeatedly
}

Example

for(let i = 1; i <= 5; i++) {
  print i
}

while Loop

let i = 1;
while(i <= 5) {
  i++
}

Key Idea

A loop runs until the condition becomes false.
