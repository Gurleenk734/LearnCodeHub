JavaScript Operators

What are Operators?

Operators are used to perform operations on variables and values, like addition, comparison, and logical decisions.

Types of Operators

  • Arithmetic → +, -, *, /
  • Comparison → ==, ===, >, <
  • Logical → &&, ||

Examples

let a = 5, b = 3;
a + b → 8
a > b → true

Important Note

== checks value only
=== checks value + type (recommended)
