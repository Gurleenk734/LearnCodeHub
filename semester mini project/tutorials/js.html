Introduction to JavaScript

What is JavaScript?

JavaScript is a programming language used to make websites interactive. It runs directly in the browser and allows you to create dynamic behavior.

What Can JavaScript Do?

  • Show alerts and messages
  • Handle button clicks
  • Update content dynamically
  • Build full web applications

alert("Hello World");
