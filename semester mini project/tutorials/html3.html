HTML Text Formatting

Text Formatting Tags

HTML provides different tags to format text like bold, italic, underline, and headings.

Common Formatting Tags

  • <b> → Bold
  • <i> → Italic
  • <u> → Underline
  • <strong> → Important (bold)
  • <em> → Emphasized (italic)

Headings

HTML provides headings from h1 to h6:

<h1>Heading 1</h1>
<h2>Heading 2</h2>

Example

<b>Bold Text</b>
<i>Italic Text</i>
<u>Underlined</u>
