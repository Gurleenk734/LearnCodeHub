Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is used to create the structure of web pages. It defines headings, paragraphs, images, and more.

Basic Structure

<html>
  <head></head>
  <body>Hello</body>
</html>

Example

<h1>Hello World</h1>
