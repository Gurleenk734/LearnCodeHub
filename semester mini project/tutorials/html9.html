HTML Semantic Tags

What are Semantic Tags?

Semantic tags clearly describe the meaning of content. They help browsers, developers, and search engines understand your webpage.

Non-Semantic vs Semantic

<div>Header</div> ❌

<header>Header</header> ✅

Common Semantic Tags

  • <header> → Top section
  • <nav> → Navigation links
  • <section> → Content section
  • <article> → Independent content
  • <footer> → Bottom section

Example Structure

<header>Header</header>
<nav>Menu</nav>
<section>Content</section>
<footer>Footer</footer>
