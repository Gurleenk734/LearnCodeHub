HTML Media

Images

Images are added using the <img> tag.

<img src="image.jpg" alt="image">

Audio

Use the <audio> tag to play sound.

<audio controls>
  <source src="audio.mp3">
</audio>

Video

Use the <video> tag to play videos.

<video width="300" controls>
  <source src="video.mp4">
</video>
