JavaScript Functions

What is a Function?

A function is a block of code that runs when called. It helps you reuse code instead of writing it again and again.

Basic Syntax

function greet() {
  console.log("Hello");
}

Calling a Function

greet();

Function with Parameters

function greet(name) {
  return "Hello " + name;
}

Example

greet("Gurleen");

Key Idea

Functions make code reusable, cleaner, and easier to manage.
